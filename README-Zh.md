## Pense

Pense这是一个的实时摄像头预览工具，支持窗口大小调整和窗口置顶功能。

## 功能

1. 窗口大小调整和窗口置顶功能。
2. 摄像头模糊、摄像头镜像、摄像头翻转功能。
3. 支持快捷操作
4. 支持调用iphone的摄像头（macos需要ventura以上）

快捷键列表：

| 快捷键 |	功能 |
|  ----  | ----  |
| command+c |	模糊窗口 |
| command+up |	放大 |
| command+down |	缩小窗口 |
| command+p |	置顶窗口 |
| command+t |	打开工具栏 |
| command+left |	将窗口移动到左上角 |
| command+right |	将窗口移动到右下角 |
| command+g |	翻转摄像头 |


## 如何关闭应用程序？

您可以使用 `command/ctrl+q` 快捷键来关闭应用程序，或者在程序栏中关闭应用程序。
